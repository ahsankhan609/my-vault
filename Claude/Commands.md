> [!important] Keep in Mind
> * use (/) before every Claude command in Claude CLI
> * use (@) to tag file in Claude CLI

| Command       | Usecase                                                                                                        | Notes                                                                                                                                                                                                                                                                                                                      |
| ------------- | -------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `claude -r`   |                                                                                                                | resume previous session                                                                                                                                                                                                                                                                                                    |
| `resume`      |                                                                                                                | resume a previous conversaton                                                                                                                                                                                                                                                                                              |
| `exit`        |                                                                                                                | exit the CLI. and close the Claude terminal                                                                                                                                                                                                                                                                                |
| `shift + !`   | use this to setup project folder, git etc...                                                                   | enter into bash command mode. it is better for context for claude                                                                                                                                                                                                                                                          |
| `shift + tab` | plan mode                                                                                                      | enter into different modes such as plan mode                                                                                                                                                                                                                                                                               |
| `rename`      |                                                                                                                | rename current session for future reference to the claude                                                                                                                                                                                                                                                                  |
| `btw`         |                                                                                                                | ask general question from Claude during working in CLI                                                                                                                                                                                                                                                                     |
| `export`      |                                                                                                                | export the current conversation to a file or clipboard                                                                                                                                                                                                                                                                     |
| `model`       |                                                                                                                | switch between models. <br>- Opus for planning project.<br>- Sonnet for implementing tasks<br>- Haiku for general questions                                                                                                                                                                                                |
| `effort`      | keep it to `medium`                                                                                            | set effort level for model usage                                                                                                                                                                                                                                                                                           |
| `usage`       |                                                                                                                | show session cost(usage), plan usage and activity stats                                                                                                                                                                                                                                                                    |
| `insights`    |                                                                                                                | generate a report analyzing your Claude code session                                                                                                                                                                                                                                                                       |
| `config`      |                                                                                                                | turn on Claude settings such as verbose, thinking mode on-off, change language                                                                                                                                                                                                                                             |
| `permissons`  | save it on project level only                                                                                  | allow Claude to do specific tasks without confirming such as doing Web search...                                                                                                                                                                                                                                           |
| `voice`       |                                                                                                                | toggle voice mode- helpful in speaking rather then writing                                                                                                                                                                                                                                                                 |
| `context`     | show current context usage as a colored grid                                                                   | see how much tokens asessiin consumed                                                                                                                                                                                                                                                                                      |
| `clear`       | start a new session, without any context.<br><br>previous session stays on disk and can be resumed with resume |                                                                                                                                                                                                                                                                                                                            |
| `compact`     | free up context  by summarizing the conversation                                                               | compact the session chat for more token room - we can also do this to make more context available for our project memory.<br><br>[compact session](https://youtu.be/eW3oTyfeWZ0?t=106&si=61UAgcI05k_LBQtT)<br><br>best practice is to run (/compact) command after every feature build. And use (/clear) if the work done. |
